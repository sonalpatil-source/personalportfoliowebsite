# Portfolio Website

A personal portfolio website built using HTML, CSS, and JavaScript to showcase my skills, projects, and technical interests in web development and software development.

---

## About

This project is a simple and responsive portfolio website designed to present my profile, technical skills, and development journey. It helps in showcasing my work in a structured and accessible way.

---

## Built With

- HTML
- CSS
- JavaScript

---

## Features

- Clean and responsive UI design
- Smooth navigation
- Sections for about, skills, and contact
- Lightweight and fast loading
- Fully static frontend project

---

## Purpose

The goal of this project is to:
- Practice frontend development fundamentals
- Build a professional online presence
- Showcase technical skills and projects
- Serve as a personal developer portfolio

---

## Note

This is an ongoing project and may be improved with additional features and better UI design over time.
